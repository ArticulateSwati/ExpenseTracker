# 💰 Expensify - Expense Tracker

A sleek, **beginner-friendly**, and highly interactive financial management tool. This project is designed to help users track their daily income and expenses with real-time balance updates and data persistence.

## ✨ Why Expensify?

This project is a perfect example of modern web logic. It is specifically crafted to be:
* 🌈 **Beginner Friendly:** Clean, commented code that is easy for anyone to read and modify.
* 💾 **Smart Storage:** Uses `localStorage` so your financial data stays saved even after closing the browser.
* ⚡ **Instant Calculations:** Uses JavaScript array methods to calculate Total, Income, and Expense totals on the fly.
* 📱 **Responsive Design:** Optimized for both mobile and desktop viewing.
